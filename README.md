## 📬 Contact Form Architecture

The contact form uses **EmailJS** for secure client-side email sending.

- No backend server required  
- No exposed SMTP credentials  
- Optimized for serverless deployment  

---

## 🧠 Custom Improvements

- Removed dependencies to make it lighter (Axios, Nodemailer)
- Simplified architecture

- Updated branding & metadata
- Improved deployment structure

```bash
git clone https://github.com/Destro-28/my-portfolio.git

npm install
npm run dev
